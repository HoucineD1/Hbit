# Hbit — Design Context

## App
Hbit — all-in-one personal growth dashboard covering Habits, Sleep, Mood, Budget, Focus, and Plan. Solo-built by Houcine (God). Freemium model targeting Francophone (Québec, France, West Africa) + English users.

## Stack
Vanilla HTML + CSS + JS, Firebase (Auth + Firestore + Realtime DB). No framework, no build tools, no npm in the app itself.

## Users
18–40, growth-oriented. Bilingual EN/FR. Mobile-first context (phone in hand, one-thumb reach). Checking in once or twice a day — not power users grinding dashboards.

## Brand Personality
**Warm. Modern. Quietly premium.** A personal coach, not a clinical tracker. Confident without shouting. The interface should feel like it respects the user's time and intelligence.

## Aesthetic Direction
- **Default:** dark theme, data-first, Oura-meets-Revolut
- **References:** Oura (subtle glow, premium materials), Apple Health (calm hierarchy, rings), Notion (sharp type, modular cards, strong empty states), Revolut (mobile finance UX, bottom sheets, FAB)
- **Anti-refs:** clinical health apps, loud gamified habit trackers, finance apps that feel like spreadsheets
- **Themes:** 4 total — dark (default), light, plus two more. Every module must pass contrast AA in all.

## Module Accents (canonical)
- Habits `#34D399` emerald
- Budget `#F59E0B` amber
- Sleep `#818CF8` indigo
- Mood `#A78BFA` violet
- Focus `#F97316` orange
- Plan `#22D3EE` cyan
- Brand red `#E63946` (auth/brand moments only)

## Design Principles
1. **Mobile first, always.** 320px minimum. Desktop is an enhancement, not the design target.
2. **Bilingual by construction.** Every visible string goes through `t()`. No literal English in module files. FR keys ship alongside EN.
3. **Tokens, not hex.** Colors, spacing, type, shadow, breakpoints all live in `css/core/tokens.css`. Modules reference, never redefine.
4. **Contrast AA minimum** across all four themes — green-on-white and muted text are the common failures.
5. **Silent failures are bugs.** Every catch surfaces a toast with retry. Empty states teach.
6. **Calm confidence.** Subtle motion, meaningful micro-interactions, no confetti carpet-bombing.
